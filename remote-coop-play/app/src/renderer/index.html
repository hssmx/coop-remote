Welcome

Choose how this computer will participate.

Host is the computer running the game. Guest is the second player watching the stream and sending keys.

Quick flow
1Open Settings only if you need to change server or TURN.
2Host creates room and shares the code.
3Guest joins and waits for approval.
4Host enables Remote input, guest locks controls.
What users need
  • Most users never need to touch server settings.
  • Use the default setup unless you are testing locally or through a tunnel.
  • The debug console below explains what is happening in real time.
