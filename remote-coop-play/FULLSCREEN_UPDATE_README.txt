FULLSCREEN UPDATE

This version adds dedicated fullscreen buttons.

Host mode:
- The Host capture panel has a Fullscreen button.
- It expands the host preview.

Guest mode:
- The Remote stream panel has a Fullscreen button.
- It expands the actual stream/control area.
- Escape exits fullscreen normally.
- If controls are locked, Escape also unlocks controls, as before.

Debug console:
- Logs when fullscreen is entered, exited, or fails.
