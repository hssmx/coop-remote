COSMIC UI UPDATE

This version redesigns the renderer into a more cosmic, product-like interface.

Main changes:
- New cosmic visual theme with a proper navigation rail
- Separate Home, Host, Join, and Settings navigation
- Connection and relay settings moved into a dedicated Settings modal
- Default server fields are now hidden from normal users
- Cleaner host and guest task flows
- Better visual hierarchy, spacing, and structure
- Existing host, guest, streaming, input, debug, and fullscreen logic preserved

What users now see:
- Home screen with quick actions and no server clutter
- Host screen only for hosting tasks
- Join screen only for joining tasks
- Settings opened only when needed
- Larger stage area and cleaner debug area
